# 💸 Expense Assistant Bot

A simple Telegram bot to track and manage your expenses automatically.

---

## 🚀 What It Does

* Log expenses from Telegram messages
* Save data to Google Sheets
* Check past expenses
* Send alerts for high spending

---

## ⚙️ How It Works

1. You send a message on Telegram
2. The bot understands it using AI
3. It stores or retrieves data from Google Sheets
4. Sends a reply back to you

---

## 🧰 Tools Used

* Telegram Bot
* Google Sheets
* AI (Gemini)
* Workflow automation tool

---

## 📂 Files

* `workflow.json` → Main workflow file
* `README.md` → Project info

---

## ▶️ Setup

1. Import `workflow.json` into your workflow tool
2. Add your:

   * Telegram Bot Token
   * Google Sheets access
   * Gemini API key
3. Activate the workflow

---

## 💬 Example

* `Add 50 for groceries`
* `Show my expenses`

---

## 🔐 Note

Keep your API keys private. Do not upload them to GitHub.
